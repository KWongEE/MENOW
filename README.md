***MENOW***

Inspired by my love for cats, I created a web application that allows the user to upload pictures of cats they find and their location as well as serving as a map to find furry friends near you.

***Tech/Framework used*** 
* React
* Rails
* Google Maps React
* Geokit Rails
* Carrierwaves
* Fog
* Devise
